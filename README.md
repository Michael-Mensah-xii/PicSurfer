## PicSurfer
Project for studying app architecture using the Unsplash API to display images from https://unsplash.com 

## Installation
* Install Android Studio on any compatible device


## Built with
* Jetpack Compose
* Compose Coil
* MVVM Architecture
* Kotlin Coroutines with Flow
* Dagger Hilt
* Paging 3
* Room for database
